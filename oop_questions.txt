1. Describe the difference between a Class and an Object.

The class is the actual code, containing the initialized variables and methods.

The objects are instances of the class.

3. Describe the difference between public and private?

public allows any class to reference the public method or variable

private will only allow the methods and variables to be accessed within their native class

5. Define the term constructor.

Constructors are methods that initialize the class variables so they can be used in the methods for instances of the class

7. Describe the purpose of overriding a method.

You can define a new behavior specific to the subclass for an abstract method from the parent.
